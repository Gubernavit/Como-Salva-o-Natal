**COMO SALVAR O NATAL** é um jogo criado em 24 horas para uma game jam interna entre alunos do senai dr. celso caruri,
e um jogo onde voce, uma rena do papai noel que se perdeu durante uma grande tempestade que espalhou todos presentes 
crianças que se comportaram o ano todo, e voce vai em busca desses presentes enquanto enfrenta os perigos que vieram 
junto a essa tempestade como bonecos de neve do mal, se aventure nessa jornada e ajude o papai noel
